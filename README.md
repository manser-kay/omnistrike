# OmniStrike — Red Team Framework

**50+ modules across 5 rarity tiers | Android/Linux/macOS | Free**

Personal Red Team framework. Evolved from a simple scanner to a digital ecosystem.

## Module Tiers

| Tier | Folder | Description | Public |
|------|--------|-------------|--------|
| Common | tools/ | Basic utilities | Yes |
| Uncommon | archive/ | Old versions | Yes |
| Rare | modules/ | Recon, SQLi, XSS, LFI | Yes |
| Epic | core/ | 12 dimensions | Yes |
| Legendary | Private | XXX, Apocalypse, C2 | No |

## Quick Start
git clone https://github.com/manser-kay/OmniStrike.git
cd OmniStrike
bash orchestrator.sh

## Install
curl -s https://raw.githubusercontent.com/manser-kay/OmniStrike/main/tools/shadow_install.sh | bash
